# 智慧医疗

开发环境
项目包含iOS端和服务器端
- ChatAi：iOS端 用的是swift语音编写
- ChatAi_go：服务器端，用go语言编写
- 服务器配置：2核4g

第三方：
- 环信IM sdk
- 同义千问 接口
