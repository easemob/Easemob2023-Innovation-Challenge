1.用Xcode直接打开HomeworkHelper.xcworkspace即可运行项目。2.通过拍照或者相册图片进行扫描快速获得题目文本（选择照片后可进行适当裁剪去除干扰内容），确认内容后即可向AI发起询问。 3.也可以选择直接输入题目对AI进行提问。
